# Fun with Computer Vision - Tutorials

Welcome! Thank you for supporting my computer vision work. Let's learn and grow together :)

This is the directory for all of my computer vision tutorials.

## Running the Code Locally

Please follow these installation instructions to download / clone all files:

```bash
# Clone this repository
git clone git@github.com:collidingScopes/fun-with-cv-tutorials.git

# Navigate to the project directory
cd fun-with-cv-tutorials

```

Now, all files are downloaded on your computer locally, saved in a folder called "fun-with-cv-tutorials".

Everything is ready for you to explore the tutorials!

## Desktop Launcher

Prefer a native experience? Launch every tutorial from a single Electron desktop app:

```bash
cd launcher
npm install
npm start
```

`npm run dist` will package the same content into platform installers (`.dmg`, `.exe`, `.AppImage`) inside `launcher/dist`.
